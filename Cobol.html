Cobol

Cobol, Stylized as COBOL, is a programming language designed to work on all platforms. It was created in the late 1950's. Cobol was designed to be business oriented, as there were no languages designed with that purpose in mind at the time. It was also the first language that had english-like syntax to become mainstream. Cobol can currantly be found in ATM and phone systems, as well as in legacy computer programs.
